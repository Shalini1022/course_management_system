# Student Course Management and Learning Progress Tracking System

##  Project Overview

The **Student Course Management and Learning Progress Tracking System** is a full-stack web application developed to simplify academic course enrollment and learning management.

The system enables students to create accounts, explore available courses, enroll in courses, access learning materials, monitor their learning progress, and receive course-related updates.

Administrators can efficiently manage students, courses, enrollments, learning progress, and generate performance reports through an admin dashboard.

This project demonstrates key concepts of **Full Stack Web Development** including frontend development, backend APIs, database management, authentication, security, and deployment.

---

# 📖 Problem Statement

In many educational institutions, students face difficulties in:

- Finding available courses
- Registering for courses efficiently
- Accessing course information in one place
- Tracking completed and pending modules
- Monitoring their learning progress

Administrators also experience challenges in:

- Managing course enrollments
- Maintaining student records
- Monitoring student performance
- Generating progress reports

The proposed system solves these issues by providing a centralized web-based learning management platform.

---

# 🎯 Project Objectives

The main objectives of this project are:

1. Provide secure student registration and login.
2. Allow students to browse and enroll in courses.
3. Enable administrators to create and manage courses.
4. Track student learning progress.
5. Generate progress reports.
6. Provide notifications and course updates.
7. Maintain a secure and user-friendly learning environment.

# ✨ Features

## 👨‍🎓 Student Features

Students can:

- Register and login securely
- Manage personal profile

- Access course materials
- Track learning progress
